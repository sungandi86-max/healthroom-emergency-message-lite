# 보건실 응급상황 연락·보고 문구 도우미 Lite

Vercel 배포를 전제로 만든 정적 웹앱입니다. 쑤캥T 보건실 도구모음에서 "앱 사용하기" 버튼으로 바로 연결해 사용하는 공개용 무료 Lite 도구입니다.

## 목적

응급상황 발생 후 보건교사가 관리자 보고, 담임 안내, 학부모 안내에 활용할 수 있는 사실 중심 문구를 빠르게 생성합니다.

이 앱은 공식 응급환자 이송 및 사고 기록지를 대체하지 않습니다. 응급처치 방법을 안내하지 않고, 의학적 판단이나 진단, 치료, 책임 판단, 안전공제 대상 여부 판단을 하지 않습니다. 실제 기록은 학교 내부 양식과 보건일지에 작성해 주세요.

## 파일 구조

- `index.html`
- `style.css`
- `app.js`
- `README.md`

## 저장 관련 안내

- 사용자 입력값을 저장하지 않습니다.
- 서버로 전송하지 않습니다.
- localStorage, sessionStorage, 쿠키, 데이터베이스를 사용하지 않습니다.
- 결과는 브라우저 화면에서만 생성됩니다.
- 사용자가 복사 버튼으로 직접 복사합니다.

## Vercel 배포

별도 빌드 과정이 필요 없는 정적 사이트입니다. 이 폴더를 Vercel 프로젝트로 연결하면 `index.html`이 기본 페이지로 배포됩니다.

## 개인정보·민감정보 입력 금지

학생 이름, 학번, 주민등록번호, 연락처, 질병명, 진단명, 검사 결과, 상담 내용, 투약 기록, 보호자 연락처, 교직원명, 개인 연락처를 입력하지 마세요.
